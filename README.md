# Word Search in File (C)

This project reads a text file and counts how many times a specific word appears.

## Features
- asks the user for a word
- reads a text file
- counts word occurrences
- displays the result

## Concepts used
- file handling
- strings
- loops
- conditions

## Input file
Create a file named `text.txt`.

## Compilation
gcc wordsearch.c -o wordsearch

## Execution
./wordsearch
